# 酒税法学習サイト

酒税法を体系的に学べるオウンドメディア。飲食店、酒販店、製造業など、酒類事業に関わる全ての人へ向けたコンテンツサイトです。

## 📋 プロジェクト概要

### 目的
- 酒税法に関する体系的な学習コンテンツを提供
- 酒類帳簿作成アプリへの自然な送客
- AIO（AI Overview）最適化されたコンテンツ設計

### サイト構成
- **基礎編**（5ページ）：酒税法とは、基本用語、FAQ
- **体系編**（7ページ）：税制体系、申告義務、改正内容
- **帳簿編**（12ページ）：実務ガイド、ベストプラクティス（コンテンツ厚め）
- **業態別編**（5ページ）：飲食店、酒販店、製造業など業種別対応

## 🚀 スタート方法

## 🎨 デザイン方針

- **色彩**：深い紺（信頼性）+ 琥珀色（酒のイメージ）
- **タイポグラフィ**：見出しはセリフ体、本文はサンセリフ体
- **レイアウト**：段階的な学習フロー（基礎→体系→帳簿→実装）を視覚的に表現
