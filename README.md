# 酒税法ラーニング｜酒類受払帳の実務サポートサイト

酒類受払帳の書き方・テンプレート・免許申請まで網羅した、小売店・飲食店向けの実務学習サイト。将来リリース予定の「酒類受払帳作成アプリ」へのメディア送客を目的とした、HubSpotモデルのオウンドメディア。

## 📋 プロジェクト概要

### 目的
- **学習サイト**：酒税法・帳簿実務の体系的な学習コンテンツ提供（2026年6月法令基準）
- **送客装置**：メディア集客 → リード獲得 → アプリ申し込みのファネルを構築
- **差別化**：概念解説だけでなく「記入例」「テンプレート」「比較表」などの実務ツール提供

### 法令根拠
- **基準日**：2026年6月
- **主要法令**：酒税法、酒税法施行規則（帳簿保存期間5年等）
- **税率改正**：2026年10月改正に対応予定

## 📝 記事執筆ガイドライン

### 法令準拠
- **基準日**：2026年6月時点
- **参考資料**：https://osake-menkyo.com/（酒販免許ドットコム）で定期的に検証
- **修正が必要な場合**：「保存期間」「記載項目」「業種別義務」を中心に確認

### コンテンツ品質
- 曖昧な記述は避ける（「〜の場合がある」は「〜の場合」に）
- 必ず**記入例・具体的な数字**を含める
- **よくある誤解**セクションで、ユーザーの実装時の迷いを先読みする
- 免責事項は必ず最後に付ける

### SEO対策
- ページタイトルは「〇〇の△△ガイド」など実務系キーワード含める
- メタ説明文（160字以内）は「何ができるか」を明示
- 見出しに実務キーワードを含める（「書き方」「テンプレート」「申請」等）
